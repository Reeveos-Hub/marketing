# ReeveOS Homepage

Zero-commission platform for UK independent businesses. Marketing homepage.

## Stack
- Vite + React
- Tailwind CSS v3
- Framer Motion
- Deploy: Vercel

## Dev
```bash
npm install
npm run dev
```

## Cursor Workflow
1. Open in Cursor
2. Figma MCP connected at `https://mcp.figma.com/mcp`
3. Paste Figma link in chat
4. Cursor reads design and builds components
